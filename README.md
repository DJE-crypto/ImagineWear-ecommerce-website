# Imagine Wear - Professional E-Commerce Website 🛒

**Imagine Wear** is a sleek, feature-rich e-commerce web application designed for a modern fashion retail experience. This project demonstrates a complete user journey, from browsing trendy apparel to managing orders and administrative product entry.

## 🚀 Key Features

* **Modern Landing Page:** Featuring seasonal collections (Autumn Sale), promotional banners, and featured products.
* **Brand Integration:** Showcases partnerships with top brands and local initiatives like **M&G Shop**.
* **Advanced Product Pages:** Detailed view with image galleries, price display in **DZD**, size selection (S, M, L, XL, XXL), and quantity controls.
* **Admin Interface:** A dedicated "Add a Product" module allowing administrators to manage the catalog (name, description, price, promo, and categories).
* **Checkout Workflow:** Professional **Order Confirmation** system providing users with order dates, expected delivery, and payment status.
* **Authentication System:** Custom-designed Login and Registration interfaces with a focus on clean UI.
* **Fully Responsive:** Built with a mobile-first approach using Bootstrap 5 to ensure compatibility across all devices.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for optimal accessibility.
* **CSS3:** Custom styling and modern layouts.
* **JavaScript (ES6+):** Logic for product selection, cart interactions, and UI components.
* **Bootstrap 5:** Powerful responsive grid system and pre-styled components.
